# 第三方库下载说明

## 一、Bootstrap 5.3.3

### 下载地址
https://getbootstrap.com/docs/5.3/getting-started/download/

### 需要下载的文件
1. **CSS文件**：`bootstrap.min.css`
   - 保存路径：`/static/vendor/bootstrap/css/bootstrap.min.css`

2. **JS文件**：`bootstrap.bundle.min.js`（含Popper.js）
   - 保存路径：`/static/vendor/bootstrap/js/bootstrap.bundle.min.js`

### 下载方式
1. 访问上述地址
2. 点击"Download"按钮
3. 解压下载的zip文件
4. 将`css/bootstrap.min.css`复制到项目目录
5. 将`js/bootstrap.bundle.min.js`复制到项目目录

---

## 二、jQuery Validation 1.19.5

### 下载地址
https://jqueryvalidation.org/

### 需要下载的文件
1. **jQuery Validation插件**：`jquery.validate.min.js`
   - 保存路径：`/static/js/jquery.validate.min.js`

2. **中文消息文件**：`messages_zh.js`
   - 保存路径：`/static/js/messages_zh.js`

### 下载方式
1. 访问上述地址
2. 点击"Download"按钮
3. 解压下载的zip文件
4. 将`dist/jquery.validate.min.js`复制到项目目录
5. 将`dist/localization/messages_zh.js`复制到项目目录

---

## 四、验证方法

下载完成后，在JSP页面中引入以下代码进行验证：

```jsp
<!-- Bootstrap CSS -->
<link rel="stylesheet" href="${pageContext.request.contextPath}/static/vendor/bootstrap/css/bootstrap.min.css">

<!-- jQuery -->
<script src="${pageContext.request.contextPath}/static/js/jquery.min.js"></script>

<!-- Bootstrap JS -->
<script src="${pageContext.request.contextPath}/static/vendor/bootstrap/js/bootstrap.bundle.min.js"></script>

<!-- jQuery Validation -->
<script src="${pageContext.request.contextPath}/static/js/jquery.validate.min.js"></script>
<script src="${pageContext.request.contextPath}/static/js/messages_zh.js"></script>
```

如果页面正常显示且无JS错误，则说明第三方库引入成功。
