# 阶段五：入住与调宿管理模块开发计划

## 一、模块概述

### 1.1 功能范围
- 入住登记：管理员/宿管为学生办理入住
- 退宿办理：管理员/宿管为学生办理退宿
- 入住信息查询：按角色查看入住记录
- 调宿申请：学生提交调宿申请
- 调宿审批：管理员/宿管审批调宿申请
- 调宿记录查询：按角色查看调宿申请

### 1.2 角色权限矩阵
| 功能 | 管理员 | 宿管 | 学生 |
|------|--------|------|------|
| 入住登记 | ✓ | ✓（本楼栋） | - |
| 退宿办理 | ✓ | ✓（本楼栋） | - |
| 入住列表 | 全部 | 本楼栋 | 本人 |
| 调宿申请 | - | - | ✓ |
| 调宿审批 | ✓ | ✓（本楼栋） | - |
| 调宿列表 | 全部 | 本楼栋 | 本人 |
| 调宿撤销 | - | - | ✓（仅待审批） |

## 六、Service层设计

### 6.1 CheckinService

```java
public interface CheckinService {
    CheckinVO getCheckinByStudentId(Long studentId);
    PageInfo<CheckinVO> getCheckinList(DormCheckinRecord query, Integer pageNum, Integer pageSize);
    PageInfo<CheckinVO> getCheckinListByBuildingId(Long buildingId, Integer pageNum, Integer pageSize);
    void checkin(CheckinDTO dto, Long operatorId);
    void checkout(Long checkinId, Long operatorId);
}
```

**业务规则**：
- checkin：
  1. 校验学生存在且状态正常
  2. 校验学生当前无在住记录
  3. 校验床位存在且状态为空闲
  4. 事务内执行：插入入住记录 + 更新床位状态为已入住
- checkout：
  1. 校验入住记录存在且状态为在住
  2. 事务内执行：更新入住记录状态为已退宿 + 更新床位状态为空闲

### 6.2 MoveApplyService

```java
public interface MoveApplyService {
    MoveApplyVO getApplyById(Long applyId);
    PageInfo<MoveApplyVO> getApplyList(DormMoveApply query, Integer pageNum, Integer pageSize);
    PageInfo<MoveApplyVO> getMyApplies(Long studentId, Integer pageNum, Integer pageSize);
    void apply(MoveApplyDTO dto, Long studentId);
    void audit(Long applyId, MoveAuditDTO dto, Long auditorId);
    void cancel(Long applyId, Long studentId);
}
```

**业务规则**：
- apply：
  1. 校验学生当前有在住记录
  2. 校验学生无待审批的调宿申请
  3. 校验目标床位存在且状态为空闲
  4. 自动获取原床位信息
  5. 插入调宿申请记录
- audit（通过时）：
  1. 校验申请状态为待审批
  2. 校验目标床位仍为空闲
  3. 事务内执行：原床位置空闲 + 目标床位置已入住 + 更新原入住记录 + 创建新入住记录 + 更新申请状态
- audit（驳回时）：
  1. 校验申请状态为待审批
  2. 更新申请状态为已驳回 + 填写审批意见
- cancel：
  1. 校验申请状态为待审批
  2. 更新申请状态为已驳回

## 七、Controller层设计

### 7.1 管理员端

**AdminCheckinController**（/admin/checkin）
- GET /list - 入住记录列表页面
- GET /page - 查询入住记录列表（分页）
- GET /building/{buildingId} - 查询楼栋入住记录
- POST /checkin - 办理入住
- POST /checkout/{checkinId} - 办理退宿

**AdminMoveController**（/admin/move）
- GET /list - 调宿申请列表页面
- GET /page - 查询调宿申请列表（分页）
- GET /pending - 查询待审批列表
- GET /{applyId} - 获取申请详情
- POST /audit/{applyId} - 审批调宿申请

### 7.2 宿管端

**DormCheckinController**（/dorm/checkin）
- GET /list - 本楼栋入住记录列表页面
- GET /page - 查询本楼栋入住记录（分页）
- POST /checkin - 办理入住
- POST /checkout/{checkinId} - 办理退宿

**DormMoveController**（/dorm/move）
- GET /list - 本楼栋调宿申请列表页面
- GET /page - 查询本楼栋调宿申请（分页）
- GET /pending - 查询待审批列表
- GET /{applyId} - 获取申请详情
- POST /audit/{applyId} - 审批调宿申请

### 7.3 学生端

**StudentCheckinController**（/student/checkin）
- GET /info - 查看本人住宿信息

**StudentMoveController**（/student/move）
- GET /list - 我的调宿申请列表页面
- GET /page - 查询我的调宿申请（分页）
- POST /apply - 提交调宿申请
- POST /cancel/{applyId} - 撤销调宿申请
