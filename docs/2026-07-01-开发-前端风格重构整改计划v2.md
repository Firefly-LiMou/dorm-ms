## 二、整改阶段

### 阶段一：index.jsp 着陆页风格重构

**目标**：将蓝色系渐变替换为杂志风暖色系，消除 hex 颜色

**涉及文件**：
- `src/main/webapp/WEB-INF/jsp/index.jsp`

**修复项**：

| 序号 | 问题 | 修复内容 |
|:---|:---|:---|
| M-1 | hex 颜色 | 将蓝色系渐变替换为杂志风暖色系（锗红/暖橙） |

**色彩方案**：
```css
/* 原蓝色系 */
background: linear-gradient(135deg, #1a73e8 0%, #0d47a1 50%, #01579b 100%);

/* 新暖色系（杂志风） */
background: linear-gradient(135deg, rgb(196, 69, 58) 0%, rgb(212, 132, 90) 50%, rgb(196, 69, 58) 100%);
```

**具体修改**：
1. `body` 背景渐变：`#1a73e8` → `rgb(196, 69, 58)`，`#0d47a1` → `rgb(212, 132, 90)`，`#01579b` → `rgb(196, 69, 58)`
2. `.btn-enter` 颜色：`#fff` → `rgb(255, 255, 255)`，`#1a73e8` → `rgb(196, 69, 58)`
3. `.btn-enter:hover` 颜色：`#0d47a1` → `rgb(176, 55, 46)`
4. 所有 `rgba(255,255,255,...)` 保持不变（已是 rgb 格式）

**验证**：
- `grep -n "#[0-9a-fA-F]\{3,8\}" index.jsp` 返回空
- 页面显示正常，暖色系渐变

**JS 联动逻辑修改**：

```javascript
// 原
$('#roomId').on('change', function() {
    var roomId = $(this).val();
    // ...
});

// 新
document.querySelector('#roomIdCselect').addEventListener('cselect:change', function(e) {
    var roomId = e.detail.value;

```
