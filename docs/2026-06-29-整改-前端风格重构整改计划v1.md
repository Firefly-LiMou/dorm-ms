# 前端风格重构整改计划

## Context

基于审查报告（阶段1-阶段2.4），发现 5 个严重问题、17 个中等问题、9 个轻微问题需要修复。本计划按依赖顺序排列，确保每个阶段的修复不会破坏已有功能。

---

## 一、整改范围

### 严重问题（5个）
- S-1: common.css 中 `--status-ok` 和 13个 `--bs-*` 变量使用 hex 格式
- S-2: bed/room/checkin/move/log 的 list.jsp 中 `.cselect` 结构错误
- S-3: login.jsp 未引入 common.css，CSS 变量未定义

### 中等问题（17个）
- M-1~M-5: 各处 hex 颜色残留
- M-6: 侧栏宽度未区分角色
- M-7: `aria-expanded` 未同步
- M-8: SVG `stroke-width` 不一致
- M-9: FontAwesome 残留
- M-10~M-11: Bootstrap 类残留
- M-12~M-13: 内联样式过多
- M-14: 头像背景应为透明
- M-15: 状态列还原为 Toggle 开关
- M-16: Bootstrap 栅格间距差异
- M-17: 缺失序号列

### 轻微问题（9个）
- L-1: CSS 章节编号重复
- L-2: 表单页原生 select
- L-3: bed/list.jsp resetSearch 缺失
- L-4~L-9: 登录页细节、首页模块卡片等

---
